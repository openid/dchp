# Code of Conduct

Participation in the Digital Credentials Harmonized Presentation (DCHP)
Working Group — including this repository, working group meetings, and the
mailing list — is governed by the OpenID Foundation's
[Groups, Activities, & Events Code of Conduct Policy](https://openid.net/wp-content/uploads/2026/05/OIDF_Groups-Activities-Events-Code-of-Conduct-Policy_Final_2026-04-30.pdf).

The current version of this and other OpenID Foundation policies can be
found on the [OpenID Foundation policies page](https://openid.net/policies/).
